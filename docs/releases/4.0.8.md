# DeepSeek Infra 4.0.8

## Legacy Frontend Retirement

4.0.8 completes the frontend migration. `/` serves the React build exclusively; `/ui/` remains a compatible React alias. `/legacy`, `DEEPSEEK_FRONTEND=legacy`, the legacy HTML/JavaScript entry, and the legacy root PWA files are removed.

The server now raises `React frontend build is missing. Run scripts/build_frontend.py.` when `static/ui/index.html` is unavailable. Android, PyInstaller, Docker, release ZIP, release smoke, preflight, and local startup all enforce the same artifact contract.

Root SPA deep links recover to the React index, while `/legacy` and `/legacy/` return 404. `/sw.js` and `/manifest.webmanifest` always resolve to React build outputs. The independent Trace Viewer, icons, fonts, and shared static resources remain available.

## Validation

```bash
npm ci --prefix frontend
npm run check --prefix frontend
python scripts/build_frontend.py
pytest tests/test_server_failure_paths.py tests/test_server_integration.py
pytest tests/test_react_frontend_contract.py tests/test_frontend_runtime_contract.py
python scripts/smoke_frontend_browser.py --out docs/evidence/frontend-browser-v4.0.8.json
python scripts/smoke_release.py --offline
python scripts/preflight_release.py --version 4.0.8 --ga
```

The frozen 4.0 protocol is unchanged. Python remains authoritative, all Rust delegates remain opt-in, and Python fallback remains available.
