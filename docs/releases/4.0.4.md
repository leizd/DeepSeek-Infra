# DeepSeek Infra 4.0.4

[中文](../../README.md) / [English](../../README.en.md)
<!-- docs-language-switcher:end -->


4.0.4 brings the React frontend to full normal-chat parity with the legacy workspace, without switching the default entry.

## Included

- Attachment pipeline: XHR upload with progress, timeout and cancellation, drag-and-drop with overlay, paste, file picker, server-driven upload limits, OCR retry, and dual-tier local image previews.
- File reader: chunked text windows with navigation, original-file mode (PDF page images, images, sandboxed frames), image lightbox with keyboard navigation, and chunk-accurate citation jumps.
- Message operations: edit-and-resend with truncation, regenerate, continue interrupted answers, copy, and single-reply Markdown export.
- History management: favorite-first ordering, inline rename with custom-title protection, and conversation search.
- Search sources and citations: `[^Wn]` web links, `[^Fn-m]` file-chunk references that reopen the reader at the cited chunk, and multi-round search blocks with expandable sources.

## Deliberately not included

- `/` does not switch to React and `static/modules/chat.js` is not deleted.
- Agent mode, Projects, Skills, Memory, speech, diagnostics, and PWA ownership remain on the legacy frontend until later parity slices.
- The frozen 4.0 runtime/protocol contract, Python-first ownership, Rust opt-in delegates, and Python fallback are unchanged.

## Verification

```bash
npm ci --prefix frontend
npm run check --prefix frontend
python scripts/smoke_release.py --offline
python scripts/preflight_release.py --version 4.0.4 --ga
```
