# DeepSeek Infra 4.0.5

<!-- docs-language-switcher:start -->
[中文](../../README.md) / [English](../../README.en.md)
<!-- docs-language-switcher:end -->


4.0.5 migrates Agent mode and the Activity panel into the React frontend, still without switching the default entry.

## Included

- Agent Run creation with presets (`full` / `auto` / plan-first), an editable plan workbench, and plan confirmation through `/api/agent-runs/{id}/plan`.
- Cursor-based stream resume: NDJSON `?after=` reconnect with stalled backoff, refresh recovery from the persisted event index, and per-phase reruns via `/api/agent-runs/{id}/rerun`.
- The full agent timeline engine: running-card merging, per-phase reasoning/output/notes, stable step ids, run summaries, and the Markdown execution report.
- Activity side panel with wide-screen auto-open, dismiss suppression, and live elapsed summaries.
- Output pause/resume as a pure client-side consumption gate shared by normal and Agent streams.

## Deliberately not included

- `/` does not switch to React and `static/modules/chat.js` is not deleted.
- Projects, Skills, Memory, speech, diagnostics, and PWA ownership remain on the legacy frontend until later parity slices.
- The frozen 4.0 runtime/protocol contract, Python-first ownership, Rust opt-in delegates, and Python fallback are unchanged.

## Verification

```bash
npm ci --prefix frontend
npm run check --prefix frontend
python scripts/smoke_release.py --offline
python scripts/preflight_release.py --version 4.0.5 --ga
```
