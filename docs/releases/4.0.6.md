# DeepSeek Infra 4.0.6

4.0.6 brings the workspace side panels and platform behaviors into the React frontend, completing feature parity ahead of the default-entry switch.

## Included

- Projects panel with document upload, reader preview, an active-project composer chip, and project attachments plus `memoryScope` merged into chat requests.
- Skills panel with search, a simplified builder, enable/disable, edit, delete, and per-project skill binding.
- Memory: enable toggle, drawer with delete/clear, and `memory_suggestion` toasts with 409-conflict replace retry.
- Diagnostics rows and a Trace waterfall (span tree, durations, errors, JSON export) from `/api/traces/{id}`.
- PWA and offline: `/ui/`-scoped service worker, installable manifest, offline send gating, and Share Target import.
- Speech read-aloud, selection quote with quote-aware prompts, and reminders with SW notifications and a management drawer.

## Deliberately not included

- `/` does not switch to React and `static/modules/chat.js` is not deleted — the entry switch ships as 4.0.7 with a `DEEPSEEK_FRONTEND=legacy` rollback.
- The frozen 4.0 runtime/protocol contract, Python-first ownership, Rust opt-in delegates, and Python fallback are unchanged.

## Verification

```bash
npm ci --prefix frontend
npm run check --prefix frontend
python scripts/smoke_release.py --offline
python scripts/preflight_release.py --version 4.0.6 --ga
```
